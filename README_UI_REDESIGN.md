## 💡 Common Tasks

### "I need to create a button"
Use: `<button class="btn btn-primary">Click Me</button>`
Reference: `UI_QUICK_REFERENCE.md`

### "What colors should I use?"
Primary: `#6B4423`
Accent: `#D4A574`
Reference: `UI_QUICK_REFERENCE.md`

### "How do I make a responsive layout?"
Use: `<div class="grid grid-3">` (auto-responsive)
Reference: `UI_QUICK_REFERENCE.md`

### "I need to add a new page"
1. Copy an existing template structure
2. Include global CSS: `<link rel="stylesheet" th:href="@{/css/global.css}">`
3. Use global classes (no inline styles)
4. Reference: `UI_DESIGN_SYSTEM.md` (Implementation)

### "I need color codes"
See: `UI_QUICK_REFERENCE.md` (Color Palette section)
